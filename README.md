## 📖 About The Project

**AlgoVisualizer** is a modern, high-performance web application built to transform the way developers prepare for technical interviews. Instead of passively reading textbook solutions, this platform brings complex computer science concepts to life through interactive visualizations, step-by-step breakdowns, and a bespoke UI/UX experience.

Whether you are mastering the sliding window technique, understanding Graph Traversals, or architecting a highly scalable rate limiter for System Design interviews, AlgoVisualizer is designed to be your ultimate companion.

## ✨ Features

- **🧠 Algorithm Visualizations:** Step-by-step visual execution of popular algorithms (Arrays, Strings, Matrices, Binary Search, Dynamic Programming, and more).
- **🏛️ System Design Hub:** A premium, glassmorphism-styled learning track covering LLD, Concurrency, Networking, Databases, and Architecture Styles.
- **🌗 Seamless Theming:** Beautifully designed Dark Mode and Light Mode with consistent accessibility and contrast standard adherence.
- **📱 Responsive UI:** Fully responsive design that works flawlessly on massive 4K monitors and mobile phones.
- **⚡ Blazing Fast:** Built with React, Vite, and TypeScript for instantaneous page loads and smooth micro-animations.

## 🛠️ Installation & Setup

Want to run AlgoVisualizer locally? Follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/algo-visualizer.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd algo-visualizer
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to `http://localhost:5173` to see the app in action!
